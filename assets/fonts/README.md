# Arabic Fonts

Place these font files here before building:

- `Amiri-Regular.ttf`
- `Amiri-Bold.ttf`
- `ScheherazadeNew-Regular.ttf`
- `ScheherazadeNew-Bold.ttf`

## Free download sources

- **Amiri** — https://github.com/aliftype/amiri/releases
- **Scheherazade New** — https://software.sil.org/scheherazade/

Both are licensed under the SIL Open Font License (free for commercial use).
